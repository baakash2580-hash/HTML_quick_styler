# HTML Quick Styler

HTML Quick Styler is a simple Python-based tool that helps users quickly generate styled HTML code.  
This project allows users to apply styles to HTML elements easily using an interactive interface.

## Features

- Generate styled HTML code quickly
- Easy-to-use interface
- Supports common HTML elements
- Quick preview of styled output
- Useful for beginners learning HTML and CSS

## Technologies Used

- Python
- Gradio
- HTML
- CSS
